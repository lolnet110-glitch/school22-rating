# Backend рейтинга Школы №22

## Как запустить

1. Открой папку `backend` в терминале.
2. Установи зависимости:

```bash
pip install -r requirements.txt
```

3. Запусти сервер:

```bash
uvicorn main:app --reload
```

4. Открой в браузере:

```text
http://127.0.0.1:8000
```

Документация API:

```text
http://127.0.0.1:8000/docs
```
